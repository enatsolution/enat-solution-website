STEP 1: Install Git
  - Download from: https://git-scm.com/download/win
  - Run installer
  - Restart computer
  - Verify: git --version

STEP 2: Create GitHub Repository
  - Go to: https://github.com
  - Click "+" ‚Üí "New repository"
  - Name: candidate-search
  - Click "Create repository"

STEP 3: Initialize Git in Project
  Command: git init

STEP 4: Configure Git
  Commands:
    git config --global user.name "Your Name"
    git config --global user.email "your.email@enatsolution.com"

STEP 5: Add Files
  Command: git add .

STEP 6: Create Commit
  Command: git commit -m "Initial commit: Candidate Search Application"

STEP 7: Add Remote
  Command: git remote add origin https://github.com/YOUR_USERNAME/candidate-search.git
  (Replace YOUR_USERNAME with your GitHub username)

STEP 8: Push to GitHub
  Commands:
    git branch -M main
    git push -u origin main

STEP 9: Enter Credentials
  - Username: Your GitHub username
  - Password: Your GitHub personal access token

================================================================================
                        üîë GET GITHUB PERSONAL ACCESS TOKEN
================================================================================

1. Go to: https://github.com/settings/tokens
2. Click "Generate new token"
3. Select "Generate new token (classic)"
4. Fill in:
   - Note: "Candidate Search Deployment"
   - Expiration: 90 days
   - Scopes: Check "repo"
5. Click "Generate token"
6. COPY THE TOKEN (you won't see it again!)
7. Use this token as your password when pushing
